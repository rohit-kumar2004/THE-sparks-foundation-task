# A basic Banking System
This project is a simple dynamic website that enables seamless money transfers between users without the need for login or user creation. It is developed as part of the web development internship with The Sparks Foundation.

## Key Features
- Seamless Money Transfers: Users can transfer money to other users quickly and efficiently.
- No Login Required: The application does not require users to create accounts or log in, ensuring a hassle-free experience.
- User-friendly Interface: The website features an intuitive and easy-to-navigate interface.
## Technologies Used
- HTML: For structuring the content of the web pages.
- CSS: For styling and enhancing the appearance of the web pages.
- JavaScript: For adding interactivity and handling dynamic functionalities.
## How to Use
1. Open the website.
2. Select the user you wish to transfer money from.
3. Choose the recipient user.
4. Enter the amount to be transferred.
5. Confirm the transaction to complete the money transfer.
